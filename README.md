# StageSync

StageSync es una aplicación para músicos en vivo que permite organizar, compartir y reproducir setlists con videos descargados.

## Objetivo

Ayudar a músicos como saxofonistas, violinistas, cantantes y grupos pequeños a preparar repertorios de forma rápida, elegante y confiable.

## Versión inicial

StageSync v0.1 será una app Android local para:

- Importar videos desde una carpeta.
- Crear biblioteca de canciones.
- Crear eventos.
- Armar setlists.
- Reordenar canciones.
- Reproducir videos en modo show.
