# StageSync - PRD

## Objetivo

StageSync es una app Android para músicos en vivo que permite organizar y reproducir setlists con videos descargados de forma rápida, elegante y confiable.

## Usuario principal

Músicos que trabajan con pistas o videos:

- Saxofonistas
- Violinistas
- Cantantes
- Tecladistas
- Dúos o grupos pequeños

## Problema

Los músicos suelen guardar sus pistas o videos en carpetas y luego deben buscarlos manualmente durante ensayos o eventos. Esto genera pérdida de tiempo, desorden y riesgo de errores durante una presentación.

## Solución

StageSync permite:

- Importar videos desde una carpeta.
- Crear una biblioteca de canciones.
- Crear eventos.
- Armar setlists.
- Reordenar canciones.
- Reproducir videos en modo show.
- Guardar historial de eventos.
- Más adelante, compartir repertorios con otros músicos.

## Versión 0.1

La primera versión será local, sin nube.

### Funciones incluidas

- Biblioteca de canciones.
- Importación desde carpeta.
- Crear eventos.
- Agregar canciones a eventos.
- Reordenar setlist.
- Modo Show.
- Reproducción manual.
- Avanzar y esperar Play.
- Reproducción automática.

## Diseño

- Tema oscuro elegante.
- Color principal dorado.
- Botones grandes.
- Interfaz simple para usar en escenario.
- Pensada para tocar con una sola mano.

## Regla principal

Todo debe ayudar al músico a ahorrar tiempo antes o durante un evento.
